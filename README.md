## Программа проверки аккаунта Samsung.
Данная программа проверяет существует ли аккаунт с веденным username,


если существует , то выводит всю информацию о нем на странице.


Способы установки:


1. Установить из Git

 ```https://github.com/NikolayVatan/samsungcheck.git```

Настройка Docker.
Перейдите в командную строку и введите следующие команду:

 ```$docker-compose up -d```


## Использование

Перейдите на веб страницу,в поле поиска напишите: 0.0.0.0:5000/search?username=<Почта пользоателя>.

Если аккаунт существует , тоу вас на странице появится информация о нем.
